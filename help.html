http_get
This block sends an HTTP GET request to the URL you specify. the URL you enter must me surrounded by quotes, or it will throw an error. Example: "http://www.your_web_site.com"
http_put
This block sends an HTTP PUT request to the URL you specify. the URL you enter must me surrounded by quotes, or it will throw an error. Example: "http://www.your_web_site.com"
